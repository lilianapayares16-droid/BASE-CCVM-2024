# BASE CCVM 2024

Este repositorio contiene el desarrollo metodológico y analítico del Trabajo Fin de Máster
enfocado en la identificación del riesgo cardiovascular a partir de datos clínicos.

## Dataset

La base de datos **BASE CCVM 2024** fue proporcionada por una Institución Prestadora de Servicios
de Salud (IPS) en Colombia. Contiene información clínica y demográfica de pacientes,
debidamente anonimizada, incluyendo variables como edad, colesterol, índice de masa corporal
(IMC) y hemoglobina glicosilada.

Por motivos de **confidencialidad, protección de datos personales y cumplimiento de la normativa
vigente (RGPD y Ley Orgánica 3/2018)**, el conjunto de datos **no se publica** en este repositorio.
El acceso a la información está restringido y sujeto a acuerdos de confidencialidad.

## Estructura del repositorio

- `data/`: carpeta reservada para datos (no incluida por razones de confidencialidad)
- `notebooks/`: notebooks de análisis exploratorio y modelado
- `src/`: scripts de procesamiento y modelado
- `docs/`: documentación adicional del proyecto
